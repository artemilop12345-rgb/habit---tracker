# Habit Tracker

Веб-приложение для отслеживания привычек, написанное на Python (Flask) с базой данных SQLite. Поддерживает многопользовательский режим: каждый пользователь видит и редактирует только свои привычки.

## Возможности
- Регистрация и вход пользователей (пароли хранятся в виде хешей, не в открытом виде)
- Добавление новых привычек
- Отметка выполнения привычки за текущий день
- Подсчёт стрика — серии дней подряд без пропуска
- Подсчёт общего количества выполнений каждой привычки
- Удаление привычек
- Изоляция данных: пользователи не могут видеть или изменять привычки друг друга

## Технологии
- Python 3
- Flask
- SQLite
- Werkzeug (хеширование паролей)
- HTML / CSS
- Git / GitHub

## Установка и запуск

```bash
pip install -r requirements.txt
python app.py
```

Открыть в браузере: http://127.0.0.1:5000
