# Habit Tracker

Веб-приложение для отслеживания привычек, написанное на Python (Flask) с базой данных SQLite.

## Возможности
- Добавление новых привычек
- Отметка выполнения привычки за текущий день
- Подсчёт общего количества выполнений каждой привычки
- Удаление привычек

## Технологии
- Python 3
- Flask
- SQLite
- HTML / CSS

## Установка и запуск

```bash
pip install -r requirements.txt
python app.py
```

Открыть в браузере: http://127.0.0.1:5000
